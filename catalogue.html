Time Sheet

Below is a dictionary-style time sheet for the action sets. Each block is split into:
1. Three sessions, e.g. "[15-10-24] Session 1: {...}" (depending on filming date)
2. Sub-sessions, e.g. "A1-A2"

Each instance corresponds to a specific scene and motion. Hence, some instances will share scenes (and performers) but not share motions.

For each instance we have labelled the start and end frame number (relevant for file names) and a description of of each instance. Other meta data (e.g. materials and speed of motion) may be added at a later date

Note: The sub-session split was done this way to minimize the number of recordings we needed to make. It consumes a lot of energy and time to boot the capture system for recording. Consequently, we have also labelled the instances between performaces where the scene changes with "Description: Scene Change". The naming is arbitrary.
